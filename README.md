# Webpack Demo
Quick demo build using webpack

## Requirements
* node `>=6.9.4`
* yarn `^0.19.1` or npm `^3.10.10`

## Getting Started

Clone the project 
```bash
$ git clone git@github.com:lucaslim/webpack-demo.git <project-name>
$ cd <project-name>
```

Install dependencies and run local server
```bash
$ yarn install
$ yarn serve
```
